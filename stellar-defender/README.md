# Stellar Defender

A high-octane 2D space shooter built with HTML5 Canvas and Vanilla JavaScript.

## How to Play
1. Open `index.html` in your web browser.
2. Click **INITIALIZE MISSION** to start.
3. Use **Arrow Keys** to move the fighter jet.
4. Press **SPACE** to shoot.
5. Destroy enemy ships to score points.
6. Avoid enemy fire and collisions!

## Features
- **Dynamic Difficulty**: The game gets faster and harder the longer you survive.
- **Visual Effects**: Particle explosions, glassmorphism UI, and parallax scrolling background.
- **Responsive Design**: Adapts to your screen size.
